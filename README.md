# 🎶 Roneat Studio Pro

**Roneat Studio Pro** is a professional digital tool designed to create, transcribe, and analyze traditional Cambodian Roneat Ek scores.

Developed by **Angel Vision Labs**, this software bridges tradition and technology — giving musicians, composers, and researchers a powerful environment to work with Khmer classical music in a modern workflow.

---

## ⭐ Quick Start Guide

### 1. Score Editor

The Score Editor is where your music comes to life.

- Type notes using numbers  
  Example:
  ```
  9 8 6 5
  ```

- Add a wave ornament using `#` after a note  
  Example:
  ```
  5#
  ```

- Insert manual bar lines using `/`

- Enable **Display Left Hand (+7 Octave)** to automatically generate the two-mallet view:
  - Right hand → Red  
  - Left hand (+7 octave) → Blue  

---

### 2. Audio AI Converter

Turn sound into sheet music in seconds.

1. Open the **Audio AI** tab  
2. Select an MP3 or WAV file  
3. Click **Generate Score with AI**

The system will:
- Analyze audio frequencies  
- Match them to the 21 Roneat blades  
- Generate a playable score automatically  

Once processing is complete, click **Import to Score Editor** to refine your transcription.

---

### 3. PDF Export

When your score is ready:

Click **Export to PDF for Printing**

The software generates a clean A4 document including:
- Khmer titles  
- Grid lines  
- Ornaments  
- Performance-ready formatting  

Perfect for rehearsal, teaching, or archiving.

---

### 4. Custom Tuning (Hz Calibration)

Every Roneat instrument has unique tuning.

In **Settings**, you can:

- Adjust the exact frequency (Hz) of each of the 21 blades  
- Match the software precisely to your physical instrument  

The Audio AI will use your custom calibration for accurate transcription.

---

## 📁 .RONEAT Project Files

During installation, Roneat Studio Pro registers the custom `.roneat` file extension.

You can now:

- Double-click any `.roneat` file  
- Automatically launch the software  
- Instantly load your saved score
