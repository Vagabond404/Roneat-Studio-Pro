# Installation & Setup

This guide covers installation on **Windows 10** and **Windows 11**.

The installer handles file setup, shortcuts, and `.roneat` file association.

### Before You Start

* **Operating system:** Windows 10 or Windows 11, 64-bit
* **Disk space:** About 500 MB
* **Permissions:** Administrator rights are recommended

{% hint style="info" %}
Download releases only from the official GitHub repository.
{% endhint %}

### Download the Installer

1. Open the releases page: [https://github.com/Vagabond404/Roneat-Studio-Pro/releases](https://github.com/Vagabond404/Roneat-Studio-Pro/releases)
2. Find the latest release.
3. Download `RoneatStudioPro_v2_Setup.exe`.

### What the Installer Configures

After installation, Windows is configured to support a smoother workflow:

* `.roneat` files use the Roneat Studio Pro icon.
* Double-clicking a `.roneat` file opens it in the app.
* The app appears in the Windows Start Menu.

### Troubleshooting

#### Windows SmartScreen

If Windows shows **"Windows protected your PC"**, click **More info**, then **Run anyway**.

This is common with newly released applications.

#### Slow first launch

The splash screen may remain visible for a few seconds on first launch.

This usually happens while the app initializes audio drivers and local resources.
