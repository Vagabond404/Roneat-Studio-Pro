```
Building from Source
For developers looking to compile Roneat Studio Pro into a standalone Windows executable (.exe) and generate the final setup file, follow these detailed instructions.
The project relies on heavy scientific and multimedia libraries, so setting up a proper build environment is crucial.
1. Environment Setup
You must have Python 3.10 or higher installed on your system.
First, clone the repository and navigate to the project root directory. Then, install the exact dependencies from the requirements file to ensure compatibility:
pip install -r requirements.txt
```
