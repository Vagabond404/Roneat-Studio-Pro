# Support & Contact

Use this page for technical help, bug reports, licensing questions, and partnership requests.

### Before You Contact Support

Start with the [FAQ](../reference/faq.md).

If the app crashes or behaves unexpectedly, check the log file first:

* **Windows log path:** `%APPDATA%\RoneatStudioPro\app.log`

This file often contains the error details needed to diagnose the problem.

### Reporting a Bug

When sending a support request, include:

1. **Software version**\
   Example: `v2.1.0`
2. **Operating system**\
   Example: `Windows 11`
3. **What happened**\
   Describe what you were doing right before the issue
4. **Relevant files**\
   Attach `app.log` and, if possible, the related `.roneat` project or source audio file

{% hint style="info" %}
Clear reproduction steps make support much faster.
{% endhint %}
